## 🛠️ Tech Stack
* **HTML5:** Structured using semantic elements for better SEO and accessibility.
* **CSS3:** Designed with **Flexbox** and **Grid** to ensure a fully responsive layout across all devices (Mobile, Tablet, Desktop).
* **JavaScript (ES6+):** Handles the core logic, event processing, and **Local Storage** integration.
* **VS Code:** The primary IDE used for development and code organization.

---

## ✨ Key Features
* **Complete Task Management:** Add, delete, and mark tasks as completed effortlessly.
* **Data Persistence:** Automatically saves tasks to the browser's **Local Storage**, ensuring data remains available after page refreshes.
* **Modern UI/UX:** A minimalist design featuring a professional color palette and intuitive interface.
* **Clean Code:** Structured for readability and future scalability, following industry best practices.
